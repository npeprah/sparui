## After Generating Each Card

### Step 1: Download Best Variation
- Generate 4 variations in Midjourney
- Upscale the best one (U1, U2, U3, or U4)
- Download high-resolution PNG

### Step 2: Post-Process (Required)
Open in Figma, Photoshop, or Photopea (free online):

1. **Resize to EXACTLY 512x768px**
   - Image → Image Size → 512 x 768 pixels
   - Maintain aspect ratio

2. **Check/Add Corner Indicators**
   - If AI missed them or they're unclear
   - Font: Orbitron Black (or similar bold geometric)
   - Value: 40px, Suit symbol: 28px
   - Color: Match suit color
   - Add text shadow for depth

3. **Enhance Kente Pattern** (if needed)
   - Create overlay layer
   - Use repeating geometric stripes
   - Set to 10-20% opacity
   - Blend mode: Multiply or Overlay

4. **Adjust Colors** (if needed)
   - Background should be warm cream, not pure white
   - Boost saturation if colors feel flat
   - Add contrast for arcade energy

5. **Add/Enhance Border**
   - 4px solid stroke in gold #FFD700
   - Add slight outer glow (4px, 20% opacity gold)

### Step 3: Export PNG
- Format: PNG-24 with transparency
- Resolution: 72 DPI
- Color space: sRGB
- Filename: temp export (will rename next)

### Step 4: Compress
1. Go to https://tinypng.com
2. Upload your PNG
3. Download compressed version
4. **Verify file size < 100KB**

### Step 5: Save to Project
```bash
# Rename based on card
mv downloaded-card.png hearts_6.png

# Move to correct folder
mv hearts_6.png /Users/nana/go/src/github.com/npeprah/sparui/frontend/public/assets/cards/hearts/
```

## Quick Troubleshooting

### Problem: AI generates photorealistic card
**Fix:** Add "flat vector illustration, cel-shaded cartoon style, NOT photorealistic"

### Problem: Suit symbol too small
**Fix:** Add "MASSIVE centered symbol occupying 70% of card height, EXTRA LARGE"

### Problem: Wrong colors (too bright or too dim)
**Fix:** List exact hex codes in prompt, add "muted warm cream background #FFF5E6"

### Problem: Busy background (pattern too strong)
**Fix:** Add "subtle MINIMAL pattern, 10% opacity, background texture only"

### Problem: No corner indicators visible
**Fix:** Don't regenerate - add them yourself in post-processing (faster)

### Problem: Wrong aspect ratio
**Fix:** Ensure you include `--ar 2:3` parameter in Midjourney

### Problem: Border missing or wrong color
**Fix:** Add in post-processing - 4px stroke, #FFD700, outside edge
