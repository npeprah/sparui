## Generation Best Practices

### Iteration Strategy
1. Generate 2-3 variations per theme
2. Select the one with best card visibility in center
3. Regenerate with prompt adjustments if needed

### Common Adjustments
- If center too dark: Add "bright center area, spotlight effect"
- If pattern too busy: Add "subtle pattern, minimal texture"
- If edges too plain: Add "ornate decorative borders, detailed corners"
- If colors too saturated: Add "muted tones, professional casino quality"

### Quality Verification
- Zoom to 100% - should look crisp
- Place sample card images on surface - must be clearly visible
- Check on mobile preview - patterns should still be distinguishable
- Verify file size after optimization (<500KB target)
