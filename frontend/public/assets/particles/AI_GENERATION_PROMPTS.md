## AI Tool Recommendations

### Midjourney
**Best for:** Stylized particles with bold outlines and consistent arcade aesthetic
**Settings:**
- Use `--style raw` for cleaner results
- Use `--v 6` for latest model
- Add `--ar 1:1` for square format
- Add `--no background` to encourage transparency

**Example command:**
```
/imagine prompt: [paste prompt above] --style raw --v 6 --ar 1:1 --no background
```

### DALL-E 3
**Best for:** Geometric shapes and precise color control
**Platform:** ChatGPT Plus or Bing Image Creator
**Tip:** Emphasize "transparent background" and "PNG format" in every prompt

### Leonardo.ai
**Best for:** Fast iteration and variations
**Settings:**
- Model: Leonardo Diffusion XL
- Preset: "Illustration" or "Anime"
- Transparency: Enable "Remove Background"
- Resolution: 512x512

### Ideogram
**Best for:** Text-free graphic elements and clean geometric designs
**Settings:**
- Style: "Design" or "3D"
- Aspect Ratio: 1:1
- Magic Prompt: Off (for precise control)

## Post-Processing Checklist

After AI generation, each texture must be:

1. **Background Removal**
   - Use Photoshop, GIMP, or remove.bg to ensure pure alpha transparency
   - No semi-transparent halos or artifacts

2. **Size Verification**
   - Exactly 512x512 pixels
   - Resize using bicubic interpolation if needed

3. **Color Correction**
   - Match exact hex codes from palette
   - Boost saturation for arcade vibrancy
   - Ensure thick black outlines are preserved

4. **File Optimization**
   - Export as PNG-24 with alpha
   - Compress using TinyPNG or pngquant
   - Target <50KB (most should be 20-30KB)

5. **Quality Check**
   - Looks crisp at 100% zoom
   - Looks good at 50% zoom (in-game size)
   - No blurry edges or compression artifacts
   - Black outlines visible and bold

## Batch Generation Strategy

**Recommended Order:**
1. Start with **Fire Effects** (most critical for "on fire" mode)
2. Generate **Explosion Effects** (used frequently for wins)
3. Create **Ice Effects** (special power mechanic)
4. Finish with **Confetti Effects** (victory celebrations)

**Generate in batches of 4:**
- Request 4 variations per prompt
- Select best option from each batch
- Maintain consistency within each category

**Quality over speed:**
- It's better to regenerate than settle for mediocre
- Aim for "Would this make someone say 'whoa'?" quality
- Each particle should capture arcade mayhem energy
