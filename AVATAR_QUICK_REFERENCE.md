## Usage Scenarios

### 1. Main Menu Profile Card
**Size:** Large (256x256px)
**Display:** Full detail, animated glow
**Example:** Player's selected avatar with name and stats

### 2. Lobby Screen Player Slots
**Size:** Medium (128x128px)
**Display:** Clear features, status indicator (ready/waiting)
**Example:** 2-4 player slots showing avatars + ready status

### 3. Game Table Player Positions
**Size:** Small (64x64px)
**Display:** Recognizable silhouette, quick identification
**Example:** Player positions around game table with timers

### 4. Leaderboards
**Size:** Small (64px) for list, Medium (128px) for top 3
**Display:** Avatar + rank + score
**Example:** Top players ranked with their avatars

### 5. Chat Messages
**Size:** Small (48-64px)
**Display:** Circular, minimal, fast loading
**Example:** Avatar next to player's chat message

## AI Generation Prompt Templates

### Universal Prompt Structure
```
[Character description], [personality] avatar portrait,
arcade game art style, cel-shaded digital illustration,

CHARACTER:
- [Age/presentation]
- [Skin tone with hex]
- [Face shape and features]
- [Expression specifics]
- [Hairstyle details]
- [Accessories]
- [Clothing]
- [Accent colors]
- Head and shoulders (bust shot)

STYLE:
- Bold black outlines (2-3px)
- Cel-shaded comic book style
- High contrast lighting
- NBA Jam / Street Fighter character portrait aesthetic
- Vibrant colors, arcade energy
- 256x256px square format
- Transparent background

MOOD: [Personality traits]

AVOID: Photorealistic, busy backgrounds, stereotypes, excessive detail
```

## Quality Validation Checklist

### Visual Quality (All 5 Avatars)
- [ ] **Expression clarity:** Each personality comes through
- [ ] **Bold outlines:** 2-3px black outlines visible on all
- [ ] **High contrast:** Arcade energy, vibrant colors
- [ ] **Cultural authenticity:** Respectful African representation
- [ ] **Recognizable:** Clear at 256px, 128px, and 64px sizes
- [ ] **Consistent style:** All 5 match cel-shaded aesthetic

### Technical Quality
- [ ] **Dimensions:** Exactly 256x256px square
- [ ] **Format:** PNG with alpha transparency
- [ ] **File size:** Each <50KB
- [ ] **Background:** Fully transparent, no artifacts
- [ ] **Color space:** sRGB
- [ ] **Resolution:** 72 DPI

### Diversity Check
- [ ] **5 distinct skin tones** (light-med to very dark)
- [ ] **5 unique hairstyles** (fade, braids, afro, locs, short+beard)
- [ ] **Gender diversity** (2 masculine, 2 feminine, 1 non-binary)
- [ ] **Age range** (20s to 50s)
- [ ] **5 personalities** (bold, calm, playful, fierce, wise)
