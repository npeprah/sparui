## General Guidelines for All Cards

### Common Elements (Include in Every Prompt)
```
African-inspired playing card design, Afro-Heritage theme,
warm cream background (#FFF5E6 to #FFE8D6 gradient),
traditional Kente cloth geometric patterns (gold, brown, tan stripes),
thick metallic gold border (#FFD700) with rounded corners (24px radius),
512x768px vertical format, PNG with transparency,
arcade game art style, high contrast, vibrant colors,
bold digital illustration, cel-shaded rendering,
professional game asset quality, clean composition
```

### Style Keywords
- Arcade game art style
- High contrast, vibrant colors
- Cel-shaded digital illustration
- Bold outlines (4px)
- Metallic gold accents
- Neon glow effects
- Modern African aesthetic
- Cultural pride without stereotypes

### What to AVOID
- Photorealistic rendering
- Excessive detail that loses clarity at small sizes
- Cultural symbols that could be appropriative
- Busy backgrounds that distract from main elements
- Low contrast or muted colors
- Generic playing card designs
- Western-centric imagery

## Batch Generation Instructions

### For Midjourney

1. **Settings:**
   - Aspect ratio: `--ar 2:3`
   - Quality: `--q 2`
   - Style: `--style raw` or `--style 4c` (test both)
   - Version: Use latest version (v6 or newer)

2. **Example Command:**
   ```
   /imagine [FULL PROMPT ABOVE] --ar 2:3 --q 2 --style raw
   ```

3. **Generate 4 variations:**
   - Review all 4 options
   - Upscale the best one (U1, U2, U3, or U4)
   - If none are perfect, use "Vary (Strong)" to get new options

4. **Save:**
   - Download as PNG
   - Resize to exactly 512x768px in image editor
   - Compress to <100KB using TinyPNG

### For DALL-E 3 (ChatGPT Plus)

1. **Settings:**
   - Size: 1024x1536 (closest to 2:3 ratio)
   - Quality: Standard or HD
   - Style: Vivid (for arcade energy)

2. **Example Command:**
   ```
   Generate this image with the following specifications:
   [PASTE FULL PROMPT ABOVE]

   Size: 1024x1536
   Style: Vivid
   Quality: HD
   ```

3. **Generate 1-2 variations:**
   - DALL-E generates 1 image at a time
   - Request variations if needed: "Generate a variation of this with [specific change]"

4. **Save:**
   - Download image
   - Resize to exactly 512x768px
   - Compress to <100KB

### For Leonardo.ai

1. **Settings:**
   - Model: Leonardo Diffusion XL or Phoenix
   - Dimensions: 512x768 (or 1024x1536, then resize)
   - Guidance Scale: 7-8
   - Number of Images: 4

2. **Generate:**
   - Paste full prompt
   - Generate 4 variations
   - Select best option

3. **Save:**
   - Download PNG
   - Verify dimensions
   - Compress if needed
