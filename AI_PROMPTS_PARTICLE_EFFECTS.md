## Generation Tips

### For Best Results:
1. **Generate 4 variations** per particle to choose the best
2. **Specify "single particle only"** if AI creates multiple
3. **Emphasize exact dimensions** in the prompt
4. **Request transparent background explicitly**
5. **Include hex color codes** for accuracy

### Common Issues & Fixes:
- **Multiple particles in one sprite:** Add "single [particle type] only" to prompt
- **Missing outlines:** Emphasize "thick black outline" or specific px width
- **Wrong colors:** Provide exact hex codes and emphasize "exact colors"
- **Too realistic:** Add "cel-shaded arcade style, NOT photorealistic"
- **Wrong dimensions:** Specify dimensions multiple times in prompt

### Post-Generation Checklist:
- [ ] Resize to exact dimensions in image editor
- [ ] Ensure transparent background (pure alpha)
- [ ] Add/adjust black outlines if needed
- [ ] Verify colors match design system
- [ ] Compress with TinyPNG (<20KB target)
- [ ] Test in Phaser at 50% scale
- [ ] Save to correct folder with correct filename

   - Best: Geometric shapes, consistent styles

3. **Leonardo.ai** ⭐⭐⭐⭐
   - Best: Fast iteration, good for variations
   - Prompt format: Use as-is, select "Game Asset" preset

- **Leonardo.ai:** Alchemy v2, Game Asset preset
- **Ideogram:** Design mode, High quality
