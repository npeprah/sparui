## Quick Reference: Card Type Differences

### Number Cards (6-10):
- VERY LARGE centered suit symbol (160px, 60% card height)
- Corner indicators with number + small suit symbol
- Metallic sheen + neon glow on center symbol
- Subtle Kente pattern (10% opacity)
- Standard 4px gold border

### Face Cards (Jack/Queen/King):
- Character portrait (60% card height)
- Corner indicators with letter + small suit symbol
- Comic book arcade art style (NOT photorealistic)
- Kente pattern 20% opacity (more visible than number cards)
- Radial gradient behind character
- Suit-specific color schemes in clothing

### Ace Cards (PREMIUM):
- EXTRA LARGE suit symbol (192px, 70% card height)
- ENHANCED 6px gold border with gradient
- Premium ornate Kente pattern (15% opacity)
- Art Deco corner flourishes
- ENHANCED neon glow (12px blur, 70% opacity)
- Gradient mesh from suit color to gold
- Maximum visual impact - most impressive in deck

## Troubleshooting Common AI Issues

### Symbol too small:
Add "MASSIVE centered symbol occupying 70% of card height, EXTRA LARGE"

### Wrong colors:
List exact hex codes, add "muted warm cream background #FFF5E6"

### Too photorealistic:
Add "flat vector illustration, cel-shaded cartoon style, NOT photorealistic"

### Pattern too busy:
Add "subtle MINIMAL pattern, 10% opacity, background texture only"

### No corner indicators:
Don't regenerate - add in post-processing (faster)

### Wrong aspect ratio:
Ensure `--ar 2:3` parameter is included
